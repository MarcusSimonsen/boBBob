# boBBob
FSharp bot for playing Scrabble

## How to run 
From the bobBob directory run: 
```dotnet run --project ./ScrabbleTemplate/```

It may take some time before the game begins, as the GADDAG needs to be built

### Achieved
- Finish a game
- Multiplayer and dictionary 
### Not achieved
- Play on all boards
- Asyncronous
